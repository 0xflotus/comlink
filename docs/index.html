Comlink

An RPC library for the web

Comlink allows you to expose JavaScript values from one realm to another ‚Äì only requiring a postMessage()-like channel.

Site

const worker = new Worker('worker.js');
const expensive = Comlink.proxy(worker);
const result = await expensive();
